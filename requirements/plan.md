## 新需求详情（2026-05-29 录入）

### 052 — 桌面宠物工作区

**问题陈述：** 用户希望岛主有桌面宠物功能，参考 codex-pet 生态，能养宠物、互动，宠物可在岛主界面和电脑桌面穿梭。

**User Story：**
> As a 岛主用户，I want 在我的数字小岛上养一只桌面宠物，它能在工作区、主界面和电脑桌面上自由活动，so that 我的数字空间更有温度和陪伴感。

**核心功能：**
1. 宠物商店 — 从 codex-pet.org 开源社区浏览/下载宠物
2. 宠物管理 — 喂食、喂水、打扮、状态系统（饥饿/口渴/心情）
3. 宠物穿梭 — 工作区内 → 岛主主界面 → 电脑桌面（PySide6 透明窗口）

**分期：**
- Phase 1：Web 界面内穿梭（CSS fixed 定位 + postMessage）
- Phase 2：PySide6 桌面透明窗口（参考 DyberPet）

**T-shirt Size：** L — 新工作区 + 桌面应用 + 开源资源集成 + 跨界面穿梭

**详细文档：** `backlog/052-desktop-pet-workspace.md`

---

### 053 — Agent 分层解决策略

**问题陈述：** Agent 处理问题时策略单一，需要分层递进机制。

**三级递进：**
1. **Level 1 — 本地优先**：工作区数据 + 技能 + 工具
2. **Level 2 — 联网兜底**：Playwright 联网（用户同意，≤5 次工具调用，≤8K tokens）
3. **Level 3 — 无限循环**：死磕模式（用户同意，token 消耗警告，安全阀 100K tokens/¥5）

**用户确认方式：** 对话中询问，用户回复确认

**T-shirt Size：** M — Agent 核心循环改造 + 用户确认交互 + 资源限制

**详细文档：** `backlog/053-agent-tiered-strategy.md`

---

### 054 — Agent Python 兜底策略

**问题陈述：** Agent 应该在常规方式和联网都解决不了时，自动想到用 Python 脚本解决。

**核心规则：**
- 自动写 Python 脚本并执行
- 安全边界：不操作 DaoZhu 文件夹以外的文件（除非用户确认）
- 执行超时 30s 自动终止
- 失败自动修正重试（最多 2 次）

**T-shirt Size：** S — 技能文件 + 已有工具复用 + 安全边界

**详细文档：** `backlog/054-python-fallback-strategy.md`

---

### 055 — 待办今日聚焦桌面侧边栏

**问题陈述：** 用户希望今日待办以桌面侧边栏形式常驻显示。

**核心功能：**
- 桌面侧边栏（PySide6），吸附屏幕边缘
- 显示今日聚焦待办，可勾选完成、快速添加
- 用户可选显示模式：桌面侧边栏 / 浏览器页面 / 关闭
- 与 052 共享"岛主桌面伴侣"PySide6 进程

**T-shirt Size：** M — 桌面应用 + API 对接 + 与 052 共享架构

**详细文档：** `backlog/055-todo-desktop-sidebar.md`

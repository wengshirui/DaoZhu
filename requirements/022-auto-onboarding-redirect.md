# REQ-022: 未配置时自动跳转引导页

> Intake ID: 2025-05-25-auto-onboarding
> 优先级: 🔴 高

---

## 问题陈述

当前引导页需要用户手动访问 /onboarding。应该是：系统检测到没有配置 API Key 时，自动跳转到引导页面。

## 用户故事

**As a** 第一次使用岛主的用户，
**I want** 打开主页时自动被引导到配置页面，
**so that** 我不会面对一个无法使用的空界面而困惑。

## 验收标准

| # | AC | 可测试性 |
|---|-----|----------|
| 1 | 无 .env 文件或 .env 中无 DEEPSEEK_API_KEY 时，访问 / 自动 302 跳转到 /onboarding | 删除 .env 后访问验证 |
| 2 | 已配置 API Key 时，访问 / 正常显示主界面 | 配置后访问验证 |
| 3 | 引导完成后写入 .env，后续访问不再跳转 | 完成引导后刷新验证 |

## T-Shirt Size

**XS (1 pt)** — 后端加一个检测逻辑

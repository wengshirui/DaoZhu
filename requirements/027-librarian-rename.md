# REQ-027: 管理员形象改名（岛管理员）

> Intake ID: 2025-05-25-librarian-rename
> 优先级: 🔴 高（品牌一致性）

---

## 问题陈述

当前叫"图书管理员"，但岛主的世界观是"数字岛屿"，应该叫**岛管理员**——管理整座岛的人，不只是图书馆。

## 修改范围

- 欢迎语："你好，我是岛管理员" （不是图书管理员）
- system prompt 中的角色描述
- README 中的相关描述
- CSS class 名可保持 `.librarian`（内部命名不影响用户）

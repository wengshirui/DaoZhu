# REQ-034: 删除类操作用户确认

> Intake ID: 2026-05-26-delete-confirm
> 优先级: 🔴 高（安全）

---

## 问题陈述

Agent 执行删除类操作（删除工作区、删除数据）时必须先获得用户确认，防止误操作。

## 规则

- 🔴 删除类操作 → 必须用户确认
- 🟢 创建/启动/停止 → 直接执行

## 验收标准

| # | AC | 可测试性 |
|---|-----|----------|
| 1 | Agent 调用 hide_workspace 前，先在对话中询问确认 | 对话验证 |
| 2 | 用户回复"确认"后才执行删除 | 流程验证 |
| 3 | 用户回复"取消"则不执行 | 取消验证 |

## T-Shirt Size

**XS (1 pt)** — Agent system prompt 中加入规则即可

# REQ-016: UI 紧急修复

> Intake ID: 2025-05-25-ui-fixes
> 优先级: 🔴 高（影响基本可用性）

---

## Bug 列表

### Bug 1: 字看不清楚

**现象**: 主界面文字辨识度不够
**原因**: 可能是字体大小、对比度、或颜色问题
**修复**: 增大基础字号 + 提高文字对比度

### Bug 2: 左右边框折叠后没法再展开

**现象**: 点击折叠侧栏后，无法再展开
**原因**: 折叠后 toggle 按钮被隐藏或不可点击
**修复**: 确保折叠状态下 toggle 按钮始终可见可点击

### Bug 3: 浏览器标签无图标

**现象**: 标签页只显示文字，没有 favicon
**修复**: 添加 favicon（像素管理员头像或岛屿 emoji）

## 验收标准

| # | AC | 可测试性 |
|---|-----|----------|
| 1 | 主界面正文字号 ≥ 14px，标题 ≥ 16px，对比度 ≥ 4.5:1 | 浏览器开发工具检查 |
| 2 | 侧栏折叠后，toggle 按钮仍可见且可点击展开 | 折叠→点击→展开 |
| 3 | 浏览器标签页显示 favicon | 视觉验证 |

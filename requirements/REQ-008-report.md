# REQ-008 报表

**状态：** ✔️ Done  
**优先级：** P1  
**T-Shirt Size：** M - 三大报表格式固定，但取数逻辑和勾稽关系需要严格

---

## 问题陈述

财务报表是企业对外披露和内部决策的核心文件。资产负债表、利润表、现金流量表（三大报表）有严格的格式和勾稽关系要求。用户需要能快速生成准确的报表。

## 用户故事

**As a** 用户  
**I want** 说"生成本月的利润表"，AI 自动从账簿取数并生成标准格式报表  
**So that** 我不需要手动汇总数据填表，也不会出现勾稽关系错误

## 范围

### In Scope

- **资产负债表** — 标准格式，资产 = 负债 + 所有者权益
- **利润表** — 标准格式，收入 - 成本 - 费用 = 利润
- **现金流量表** — 直接法/间接法（先支持直接法）
- **科目余额表** — 全科目期初/发生/期末
- **报表导出** — 导出为 Excel/PDF 格式
- **期间对比** — 本期 vs 上期、本期 vs 去年同期

### Out of Scope

- 合并报表
- 审计调整
- 自定义报表模板设计器（后续迭代）

## 验收标准

1. **AC-1：资产负债表平衡** — 生成的资产负债表必须满足 资产合计 = 负债合计 + 所有者权益合计，不平衡时报错并提示原因
2. **AC-2：利润表取数** — 利润表各项目从对应损益科目的本期发生额取数，与明细账可追溯核对
3. **AC-3：期间对比** — 用户说"对比一下今年和去年的利润表"，Agent 生成双列对比报表，标注增减变动
4. **AC-4：导出功能** — 报表可导出为 Excel 文件，格式规范（含表头、单位、编制日期）
5. **AC-5：零值处理** — 金额为零的行项目默认隐藏，用户可选择显示全部

## 业务规则

- 资产负债表是时点报表（某一天的余额）
- 利润表是期间报表（一段时间的累计发生额）
- 现金流量表是期间报表
- 报表项目与科目的对应关系需要可配置（不同准则对应关系不同）

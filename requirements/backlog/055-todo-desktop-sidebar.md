# 055 — 待办今日聚焦桌面侧边栏

> 状态: 🆕 待开发
> 优先级: P3
> T-shirt Size: M — 桌面应用（PySide6/Electron）+ 与待办工作区 API 对接 + 用户可选显示模式；风险在于桌面应用跨平台兼容
> 录入日期: 2026-05-29

---

## 问题陈述

用户希望"今日聚焦"的待办事项能以桌面侧边栏形式常驻显示，不需要打开浏览器就能看到当天要做的事。同时允许用户选择显示方式（桌面侧边栏 / 浏览器面板 / 系统托盘）。

## 范围

**In Scope:**
- 桌面侧边栏应用，常驻显示今日待办
- 用户可选择显示模式：桌面侧边栏 / 浏览器独立页 / 系统托盘弹窗
- 与待办工作区 API 实时同步
- 支持在侧边栏内勾选完成、快速添加
- 最小化/自动隐藏

**Out of Scope:**
- 完整的待办管理功能（那是工作区的事）
- 跨设备同步
- 日历视图

---

## User Story

> As a 岛主用户，I want 今日聚焦的待办以桌面侧边栏形式常驻在屏幕边缘，so that 我不用打开浏览器就能随时看到今天要做什么、快速勾选完成。

---

## 验收标准

1. 提供桌面侧边栏应用，可吸附在屏幕左/右边缘
2. 显示今日聚焦的待办列表（标题 + 优先级 + 截止时间）
3. 可在侧边栏内勾选完成，实时同步到待办工作区
4. 可在侧边栏内快速添加新待办（输入框 + 回车）
5. 支持自动隐藏（鼠标移开后收起，移到边缘时展开）
6. 用户可在设置中选择显示模式：桌面侧边栏 / 浏览器页面 / 关闭
7. 侧边栏样式与岛主整体视觉风格一致
8. 待办工作区未启动时，侧边栏显示"待办服务未运行"并提供启动按钮

---

## 技术方案

### 方案 A：PySide6 桌面应用（推荐）

- 透明无边框窗口，吸附屏幕边缘
- 通过 HTTP API 与待办工作区通信（`GET /api/todos?focus=today`）
- 可与桌面宠物（052）共享 PySide6 进程
- 轮询或 WebSocket 实时更新

### 方案 B：浏览器独立页面（降级方案）

- 提供 `/sidebar` 路由，精简版今日聚焦页面
- 用户可用浏览器"画中画"或侧边栏扩展固定
- 无需额外桌面应用

### 共享架构（与 052 桌面宠物）

桌面宠物和待办侧边栏都需要 PySide6 桌面进程，可合并为一个"岛主桌面伴侣"应用：
- 系统托盘图标（右键菜单：显示宠物/显示侧边栏/设置/退出）
- 宠物透明窗口
- 侧边栏窗口
- 统一与岛主 Web 服务通信

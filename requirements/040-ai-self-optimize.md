# REQ-040: AI 自我优化机制

> Intake ID: 2026-05-26-ai-self-optimize
> 优先级: 🟡 中

---

## 问题陈述

Agent 没有根据日志和聊天记录自我优化的机制。同样的错误会反复出现，不会从失败中学习。

## 用户故事

**As a** 岛主平台，
**I want** Agent 能从历史失败中学习并优化行为，
**so that** 越用越聪明，不重复犯同样的错误。

## 设想

1. 定期分析 skill_usage 中失败率高的工具调用
2. 从失败的对话中提取"教训"存入 knowledge 表
3. 下次遇到类似场景时，注入历史教训到 context
4. 自动优化 skill 文件（如补充注意事项）

## 验收标准

| # | AC |
|---|-----|
| 1 | Agent 调用工具失败后，自动记录失败原因到 knowledge 表 |
| 2 | 下次类似请求时，context 中包含历史失败教训 |
| 3 | 连续 3 次相同模式的失败后，Agent 主动建议用户调整 |

## T-Shirt Size

**M (5 pts)** — 失败记录 + knowledge 注入 + 模式识别

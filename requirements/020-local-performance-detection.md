# REQ-020: 本地性能检测 + 智能推荐

> Intake ID: 2025-05-25-local-performance
> 优先级: 🟢 低

---

## 问题陈述

用户电脑性能各异。如果有不错的显卡，可以建议用本地模型做语音解析、图片解析，省钱又快。

## 用户故事

**As a** 岛主用户，
**I want** 平台自动检测我的电脑性能并推荐最优配置，
**so that** 有好显卡时用本地模型，没有时用云端 API。

## 验收标准

| # | AC |
|---|-----|
| 1 | 首次启动检测 GPU 型号和显存 |
| 2 | 显存 ≥ 8GB 时建议使用本地 Ollama |
| 3 | 提供一键安装 Ollama + 推荐模型的引导 |

## T-Shirt Size

**S (3 pts)** — 检测 + 推荐 UI，不含实际本地模型集成

# REQ-030: README 展示页增加"打开工作区"按钮

> Intake ID: 2025-05-25-readme-open-btn
> 优先级: 🟡 中

---

## 问题陈述

用户单击工作区看到 README 后，想打开这个工作区，但需要返回再双击。应该在 README 展示页右上角同时提供"返回聊天"和"打开工作区"两个按钮。

## 验收标准

| # | AC | 可测试性 |
|---|-----|----------|
| 1 | README 展示页顶部有"← 返回聊天"和"▶ 打开工作区"两个按钮 | 视觉验证 |
| 2 | 点击"打开工作区"启动并在新标签打开 | 点击验证 |
| 3 | 工作区已运行时直接打开，未运行时先启动 | 两种状态验证 |

## T-Shirt Size

**XS (1 pt)** — 在现有 ReadmeViewer 中加一个按钮

# REQ-011: PyInstaller 打包发布

> Intake ID: 2025-05-25-packaging

---

## 原始需求

将岛主平台打包为单个 exe 文件，普通用户双击即可运行，无需安装 Python 环境。

---

## 提取意图

- PyInstaller 打包平台核心为单 exe
- 包含内嵌 Python 运行时
- 首次启动自动创建 workspaces/ 目录
- 系统托盘图标 + 开机自启（可选）
- Windows 安装向导（可选，NSIS/Inno Setup）

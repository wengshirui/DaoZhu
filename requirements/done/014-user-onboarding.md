# REQ-014: 用户引导页面

> Intake ID: 2025-05-25-user-onboarding
> 优先级: 🔴 高（不做引导用户根本用不起来）

---

## 问题陈述

用户下载岛主后，不知道怎么配置 API Key、不知道怎么开始用。没有引导 = 没有用户。

## 用户故事

**As a** 第一次打开岛主的用户，
**I want** 看到清晰的引导流程告诉我怎么配置和使用，
**so that** 我能在 5 分钟内开始和管家对话。

## 引导内容

1. **欢迎页** — 岛主是什么，30 秒能理解
2. **配置 DeepSeek API Key** — 手把手教注册 + 获取 Key + 填入
3. **配置 Gitee 账号**（可选）— 如果需要市场功能
4. **第一次对话** — 引导用户说第一句话（如"帮我看看有什么工作区"）
5. **完成** — 恭喜，你的岛已经建好了

## 验收标准

| # | AC | 可测试性 |
|---|-----|----------|
| 1 | 首次启动（无 config.json）自动显示引导页面 | 删除 config 后启动验证 |
| 2 | 引导页提供 DeepSeek 注册链接和 API Key 获取步骤截图 | 视觉验证 |
| 3 | 用户填入 API Key 后，引导页自动验证连通性 | 填入后显示"✅ 连接成功" |
| 4 | 引导完成后自动跳转到主界面 | 流程走完验证 |
| 5 | 用户可随时从设置中重新进入引导 | 点击设置→重新引导 |

## T-Shirt Size

**S (3 pts)** — 纯前端页面 + 配置写入，无复杂逻辑

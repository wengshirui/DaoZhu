# REQ-031: 设置页面（API Key + Gitee 配置）

> Intake ID: 2025-05-25-settings-page

## 问题陈述

当前只有引导页能配置 API Key，之后想修改就没入口了。需要一个设置页面，能配置 DeepSeek Key、Gitee Token 等信息。

## 验收标准

| # | AC | 可测试性 |
|---|-----|----------|
| 1 | 顶栏设置按钮(⚙️)点击后显示设置面板 | 点击验证 |
| 2 | 设置面板可修改 DeepSeek API Key（密码输入框） | 修改后验证连通性 |
| 3 | 设置面板可配置 Gitee Token | 填入后论坛可发帖 |
| 4 | 设置面板可切换主题（亮/暗） | 切换验证 |
| 5 | 修改后立即生效，无需重启 | 修改后测试 |

## T-Shirt Size

**S (3 pts)** — 前端面板 + 已有 config API

# REQ-006: 平台级 AI Agent

> Intake ID: 2025-05-25-platform-agent

---

## 原始需求

平台需要一个 AI Agent 作为"管家"，理解用户自然语言指令，协调技能和工具完成任务。参考 Hermes-Agent 的 AIAgent 类（对话循环 + 工具调用）。

---

## 提取意图

agent.py 是平台的 AI 核心，负责：
- 接收用户消息，调用 LLM 理解意图
- 根据意图选择合适的技能（Skill）执行
- 调用工具（MCP）完成具体操作
- 维护对话上下文
- 流式返回响应给前端

---

## 核心架构（参考 Hermes）

```
用户消息 → Agent 对话循环
              │
              ├─ LLM 推理（意图识别）
              │
              ├─ 工具调用（MCP: playwright, filesystem...）
              │
              ├─ 技能执行（create-workspaces, create-skill...）
              │
              └─ 响应生成 → 返回前端
```

### 与 Hermes-Agent 的对比

| 维度 | Hermes-Agent | 岛主 Agent |
|------|-------------|-----------|
| 定位 | 通用 AI 助手 | 工作区建造专家 |
| 工具 | 50+ 工具 | 精简核心工具 |
| 技能 | 可扩展技能系统 | 同样可扩展 |
| 对话 | CLI + 多平台 | Web 聊天窗口 |
| 模型 | 多提供商 | DeepSeek/OpenAI |

# REQ-023: 工作区软删除（隐藏）功能

> Intake ID: 2025-05-25-workspace-soft-delete
> 优先级: 🟡 中

---

## 问题陈述

用户想删除不需要的工作区，但当前没有删除入口。需要提供"隐藏"功能（软删除），真正删除由用户自行删文件夹。

## 用户故事

**As a** 岛主用户，
**I want** 能在界面上隐藏不需要的工作区，
**so that** 书架保持整洁，但数据不会丢失。

## 验收标准

| # | AC | 可测试性 |
|---|-----|----------|
| 1 | 工作区卡片右键或长按出现"隐藏"选项 | 右键菜单验证 |
| 2 | 点击隐藏后工作区从书架消失，workspace.json 中 hidden=true | 检查文件 |
| 3 | 设置中有"显示已隐藏工作区"开关，打开后可取消隐藏 | 设置页验证 |
| 4 | 隐藏操作有确认提示："隐藏后可在设置中恢复，文件不会删除" | 弹窗验证 |

## T-Shirt Size

**S (3 pts)** — 前端交互 + 已有 hide/unhide API

## 备注

后端 API 已实现（POST /api/workspaces/{id}/hide 和 /unhide），只需前端对接。

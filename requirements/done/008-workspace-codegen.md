# REQ-008: 工作区代码生成

> Intake ID: 2025-05-25-workspace-codegen

---

## 原始需求

管家需要能从零生成完整工作区代码：FastAPI 后端 + 前端页面 + SQLite schema + workspace.json。这是 create-workspaces 技能的核心执行能力。

---

## 提取意图

当搜索开源项目无果时，Agent 需要：
1. 根据用户需求设计数据模型（SQLite schema）
2. 生成 FastAPI 路由（CRUD + 业务逻辑）
3. 生成前端页面（HTML + CSS + JS）
4. 生成 workspace.json 元信息
5. 创建 venv 并安装依赖
6. 初始化数据库

参考 Hermes-Agent 的代码生成能力（通过 LLM + 工具链实现）。

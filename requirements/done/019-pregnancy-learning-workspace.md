# REQ-019: 孕期管理 + 小孩学习辅助工作区

> Intake ID: 2025-05-25-pregnancy-learning
> 优先级: 🟢 低（需要 iOS 兼容，后续迭代）

---

## 问题陈述

用户有孕期管理和小孩学习辅助的需求。小孩学习可以通过对话、上传图片了解进度和优缺点。

## 功能设想

### 孕期管理
- 孕周计算 + 产检提醒
- 体重/血压记录
- 胎动计数
- 营养建议

### 小孩学习辅助
- 对话式学习（问答）
- 上传作业图片 → AI 分析优缺点
- 学习进度追踪
- 薄弱点识别

## 约束

- 需要兼容 iOS（可能需要 PWA 或响应式 Web）
- 图片上传需要 Vision API
- 优先级低，等核心功能稳定后再做

## T-Shirt Size

**L (8 pts)** — 多功能 + 图片分析 + 跨平台

# REQ-021: 资源 README 展示

> Intake ID: 2025-05-25-resource-readme-display
> 优先级: 🔴 高

---

## 问题陈述

用户在左侧面板看到建筑/工具/技能列表，但点击后不知道它们是干什么的。需要点击后在主界面展示对应的 README 内容，让用户了解每个资源的用途。

## 用户故事

**As a** 岛主用户，
**I want** 点击左侧的建筑/工具/技能时看到它的说明文档，
**so that** 我知道每个资源是做什么的、怎么用。

## 验收标准

| # | AC | 可测试性 |
|---|-----|----------|
| 1 | 点击建筑（工作区）卡片，中间区域显示该工作区的 README.md 内容（Markdown 渲染） | 点击 todo → 显示待办 README |
| 2 | 点击技能卡片，中间区域显示该 skill 的 SKILL.md 内容 | 点击 create-workspaces → 显示 SKILL.md |
| 3 | 点击工具卡片，中间区域显示工具的描述和参数说明 | 点击 list_workspaces → 显示描述 |
| 4 | README 展示区有"返回聊天"按钮，点击回到对话界面 | 点击返回 → 回到聊天 |

## T-Shirt Size

**S (3 pts)** — 前端展示 + API 读取文件内容

# 小企业会计准则 — 现金流量表编制指南

> 参考依据：《小企业会计准则》（财会[2011]17号）第八十二条至第八十五条

---

## 一、现金及现金等价物范围

| 科目代码 | 名称 | 说明 |
|----------|------|------|
| 1001 | 库存现金 | 企业持有的纸币和硬币 |
| 1002 | 银行存款 | 可随时用于支付的活期存款 |
| 1012 | 其他货币资金 | 银行汇票存款、信用卡存款等 |

### 现金等价物（JSON 中未单独列示）

以下项目视同现金，编制时应纳入现金范围：

- 三个月内到期的国债
- 三个月内到期的银行承兑汇票
- 持有期限短、流动性强、易于转换为已知金额现金且价值变动风险很小的短期投资

**注意**：不能随时支取的定期存款不属于现金范围。

## 二、编制步骤

### 步骤 1：准备基础数据

1. 收集期末和期初的资产负债表、利润表
2. 获取现金、银行存款日记账和总账
3. 重点关注现金类科目（1001/1002/1012）的借贷方发生额
4. 确认哪些短期投资属于现金等价物

### 步骤 2：填列经营活动现金流量

采用直接法，逐项分析现金收支：

**流入项：**
- 销售商品收到的现金 = 主营业务收入 + 销项税额 + 应收账款减少额 − 应收账款增加额
- 收到税费返还 = 实际收到的退税金额
- 收到其他经营现金 = 政府补助 + 押金返还 + 罚款收入等

**流出项：**
- 购买商品支付的现金 = 主营业务成本 + 进项税额 + 存货增加额 + 应付账款减少额 − 应付账款增加额
- 支付职工现金 = 应付职工薪酬借方发生额（含社保公积金）
- 支付税费 = 应交税费借方发生额（实际缴纳部分）
- 支付其他经营现金 = 管理费用/销售费用中的现金支出部分（扣除工资、折旧等非现金项）

### 步骤 3：填列投资活动现金流量

- 处置固定资产收到现金 = 处置收入 − 清理费用（运输费、拆卸费等）
- 购建固定资产支付现金 = 从在建工程/固定资产科目借方取（不含利息资本化部分）

### 步骤 4：填列筹资活动现金流量

- 吸收投资收到现金 = 实收资本/资本公积贷方中的现金投入
- 借款收到现金 = 短期/长期借款贷方发生额
- 偿还债务支付现金 = 借款科目借方发生额（本金）
- 分配利润支付现金 = 应付利润/应付股利借方发生额

### 步骤 5：计算净增加额并验证

```
现金净增加额 = 经营活动净额 + 投资活动净额 + 筹资活动净额
期末现金余额 = 期初现金余额 + 现金净增加额
```

**验证**：期末现金余额必须与资产负债表中货币资金项目一致。如不一致，核对银行对账单。

### 步骤 6：审核与披露

- 交叉验证与资产负债表现金变动一致
- 检查外币现金流量折算（汇率变动影响单独列示）
- 附注中说明重要项目（如关联方交易现金流量）

## 三、增值税处理规则

| 场景 | 处理方式 | 归属活动 |
|------|----------|----------|
| 销项税额 | 计入经营活动现金流入 | 经营 |
| 进项税额 | 计入经营活动现金流出 | 经营 |
| 增值税出口退税 | 计入"收到的税费返还" | 经营 |
| 购建固定资产的进项税 | 计入投资活动现金流出 | 投资 |

**一般纳税人注意**：退税的现金流入应单独在"收到的税费返还"中反映。

## 四、利息和股利的分类

| 项目 | 归属活动 | 说明 |
|------|----------|------|
| 利息支出（经营借款） | 经营活动流出 | 从财务费用科目取 |
| 利息支出（资本化） | 筹资活动流出 | 计入"偿还借款利息支付的现金" |
| 利息收入 | 经营活动流入 | 银行存款利息等 |
| 股利支付 | 筹资活动流出 | 计入"分配利润支付的现金" |
| 股利收入 | 投资活动流入 | 计入"取得投资收益收到的现金" |

## 五、间接法验证公式（补充资料）

虽然小企业准则不强制要求编制间接法补充资料，但可用于验证经营活动净额：

```
经营活动现金流量净额 = 净利润
    + 折旧/摊销
    + 资产减值损失
    + 固定资产报废损失
    + 财务费用（利息支出）
    − 投资收益
    + 存货减少（或 − 存货增加）
    + 经营性应收项目减少（或 − 增加）
    + 经营性应付项目增加（或 − 减少）
```

## 六、常见问题与解决方案

### Q1：非现金交易如何处理？

现金流量表只记录现金收付。非现金交易（赊销、以物抵债、资产置换）不填入任何活动类别。

**处理方式**：
- 忽略非现金部分，仅调整影响现金的项目
- 在报表附注中披露重大非现金交易（如"以设备抵债 10,000 元"）

### Q2：营运资本变动调整困难？

**公式**：
```
调整后净额 = 净利润 + 折旧 − 存货增加 − 应收增加 + 应付增加
```

注意应付账款减少视为现金流出增加（需加回为负数）。

### Q3：小额现金（备用金）遗漏？

备用金支付的差旅费、办公费等应计入经营活动"支付的其他与经营活动有关的现金"。

**解决方案**：全面扫描现金日记账，分类汇总，使用会计软件可减少人为遗漏。

### Q4：期末现金余额与资产负债表不一致？

常见原因：
1. 未识别的现金等价物（如三个月内到期的国债未纳入）
2. 外币存款汇率变动未调整
3. 银行未达账项

**解决方案**：
- 核对银行对账单，编制余额调节表
- 外币折算差额计入"汇率变动对现金的影响"行
- 确认现金等价物范围是否完整

### Q5：利息支付应归入哪个活动？

小企业准则下：
- 经营性借款利息 → **经营活动**流出（"支付其他与经营活动有关的现金"或单列）
- 资本化利息 → **筹资活动**流出（"偿还借款利息支付的现金"）

### Q6：政府补助如何分类？

- 与日常经营相关的政府补助 → **经营活动**流入（"收到其他与经营活动有关的现金"）
- 与购建长期资产相关的政府补助 → **投资活动**流入

## 七、最佳实践

1. **定期培训**：会计人员应熟悉小企业会计准则第31号（现金流量表）
2. **使用标准模板**：采用财政部标准格式，避免格式错误
3. **保留原始凭证**：银行回单、发票等作为编制依据
4. **避免常见陷阱**：
   - 不要将利润表项目直接复制到现金流量表
   - 注意权责发生制与收付实现制的转换
   - 区分经营活动与投资/筹资活动的边界
5. **分析现金流量比率**：
   - 经营活动净额 / 流动负债 → 评估短期偿债能力
   - 经营活动净额 / 净利润 → 评估盈利质量
6. **软件辅助**：使用用友、金蝶等会计软件的现金流量模块可自动归集

`cash_flow.json` 定义了报表的行项目结构、公式和科目映射，用于程序化生成现金流量表。本指南作为补充文档，提供：

- 编制步骤的操作指导
- 特殊场景的处理规则
- 常见错误的排查方法
- 验证与审核的最佳实践

两者配合使用：JSON 驱动自动计算，MD 指导人工判断和异常处理。

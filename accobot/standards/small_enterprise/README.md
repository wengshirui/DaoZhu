# 小企业会计准则

**全称：** 《小企业会计准则》（财会〔2011〕17号）  
**适用范围：** 在中华人民共和国境内依法设立的、符合《中小企业划型标准规定》的小型企业和微型企业

## 核心规则

1. 不要求计提坏账准备（应收账款实际发生损失时直接核销）
2. 不使用递延所得税资产/负债科目
3. 不要求资产减值准备（固定资产、无形资产等不计提减值）
4. 不使用公允价值变动损益科目
5. 长期股权投资统一采用成本法
6. 财务报表较简化（无现金流量表强制要求）
7. 政府补助收到时直接计入营业外收入

## Agent 做账约束

- 禁止使用科目：递延所得税资产、递延所得税负债、坏账准备、存货跌价准备、固定资产减值准备、无形资产减值准备、资产减值损失、公允价值变动损益
- 应收账款坏账处理：直接借记营业外支出，贷记应收账款
- 固定资产处置：不涉及减值准备的转出
- 所得税费用 = 当期应交所得税（不做递延处理）
- 政府补助：收到时全额计入营业外收入（不区分与资产/收益相关）

## 报表要求

- 资产负债表（简化格式）
- 利润表（简化格式）
- 现金流量表（可选，鼓励编制）
- 附注（简化披露）

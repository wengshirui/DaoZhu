# 凭证模板使用说明

本目录包含小企业会计准则下常见业务场景的标准分录模板。

## 模板结构

每个 JSON 文件包含一组模板，每个模板包含：
- `id`: 唯一标识
- `name`: 业务场景名称
- `keywords`: 关键词列表（用于匹配用户描述）
- `entries`: 标准分录（account_code + direction）
- `note`: 使用说明/注意事项

## 使用方式

Agent 收到用户的记账请求时：
1. 提取用户描述中的关键词
2. 匹配凭证模板的 keywords 字段
3. 匹配成功 → 按模板生成分录（用户只需提供金额）
4. 匹配失败 → 回退到模型推理

## 文件分类

- `income.json` — 收入类（销售商品、提供服务等）
- `expense.json` — 支出类（房租、水电、办公用品等）
- `salary.json` — 薪酬类（工资、社保、公积金等）
- `tax.json` — 税费类（增值税、附加税、所得税等）
- `period_end.json` — 期末类（结转损益、折旧摊销等）

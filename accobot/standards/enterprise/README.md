# 企业会计准则

**全称：** 《企业会计准则》（财政部令第76号及后续修订）  
**适用范围：** 上市公司、大中型企业、以及自愿执行的其他企业

## 核心规则

1. 要求计提坏账准备（预期信用损失模型）
2. 使用递延所得税资产/负债科目
3. 要求资产减值测试和计提减值准备
4. 使用公允价值变动损益科目
5. 长期股权投资区分成本法和权益法
6. 要求编制完整现金流量表
7. 要求更详细的附注披露

## Agent 做账约束

- 应收账款需按预期信用损失模型计提坏账准备
- 每期末需评估递延所得税资产/负债
- 固定资产、无形资产需进行减值测试
- 交易性金融资产按公允价值计量，变动计入公允价值变动损益
- 长期股权投资需区分成本法（控制）和权益法（重大影响）
- 政府补助需区分与资产相关和与收益相关，分别处理
- 期末需编制现金流量表

## 报表要求

- 资产负债表（完整格式）
- 利润表（完整格式）
- 现金流量表（必须）
- 所有者权益变动表
- 附注（详细披露）

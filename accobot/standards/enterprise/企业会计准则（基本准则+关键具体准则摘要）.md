# 企业会计准则

**文号：** 财政部令第76号（基本准则）  
**施行日期：** 2007年1月1日（后续持续修订）  
**参考来源：** kjxy.hbue.edu.cn（湖北经济学院会计学院）

---

## 基本准则（第76号令）核心条款

### 第一章 总则

- 适用于在中华人民共和国境内设立的企业（包括公司）
- 上市公司、大中型企业必须执行

### 第二章 会计信息质量要求

- 可靠性、相关性、可理解性、可比性
- 实质重于形式
- 重要性原则
- 谨慎性原则（不应高估资产或收益，不应低估负债或费用）

### 第三章 资产

- 资产应当按照取得时的实际成本计量
- 符合条件的可以按公允价值计量
- **需要计提资产减值准备**（与小企业准则的核心差异）

### 第四章 负债

- 负债按实际发生额计量
- 或有负债需要披露

### 第五章 所有者权益

- 实收资本、资本公积、盈余公积、未分配利润
- 其他综合收益（小企业准则无此概念）

### 第六章 收入

- 收入确认需满足五步法模型（新收入准则 CAS 14）
- 区分在某一时段内履行和在某一时点履行的履约义务

### 第七章 费用

- 费用按功能分类
- 资产减值损失单独列示

### 第八章 利润

- 营业利润 = 营业收入 - 营业成本 - 税金及附加 - 销售费用 - 管理费用 - 财务费用 - 资产减值损失 + 公允价值变动收益 + 投资收益
- 利润总额 = 营业利润 + 营业外收入 - 营业外支出
- 净利润 = 利润总额 - 所得税费用

## 关键具体准则摘要

### CAS 8 资产减值

- 企业应当在资产负债表日判断资产是否存在减值迹象
- 存在减值迹象的，应当估计其可收回金额
- 可收回金额 = max(公允价值-处置费用, 预计未来现金流量现值)
- **长期资产减值损失一经确认，不得转回**
- 应收账款坏账准备可以转回

### CAS 18 所得税

- 采用资产负债表债务法
- 识别暂时性差异（资产/负债的账面价值与计税基础之差）
- 可抵扣暂时性差异 → 递延所得税资产
- 应纳税暂时性差异 → 递延所得税负债
- 所得税费用 = 当期所得税 + 递延所得税费用

### CAS 22 金融工具确认和计量

- 金融资产分类：以摊余成本计量、以公允价值计量且变动计入其他综合收益、以公允价值计量且变动计入当期损益
- 交易性金融资产按公允价值计量

### CAS 2 长期股权投资

- 控制 → 成本法（编制合并报表时调整）
- 共同控制/重大影响 → 权益法
- 无重大影响 → 按金融工具准则处理

### CAS 4 固定资产

- 折旧方法：年限平均法、工作量法、双倍余额递减法、年数总和法
- 需要进行减值测试
- 折旧方法、使用寿命、预计净残值至少每年复核

### CAS 6 无形资产

- 研究阶段支出 → 费用化
- 开发阶段支出（满足条件）→ 资本化
- 使用寿命有限 → 摊销
- 使用寿命不确定 → 不摊销，但每年减值测试

### CAS 14 收入（2017修订）

- 五步法模型：
  1. 识别合同
  2. 识别履约义务
  3. 确定交易价格
  4. 分摊交易价格
  5. 确认收入（履行履约义务时）

### CAS 16 政府补助

- 与资产相关 → 确认为递延收益，在资产使用寿命内分摊
- 与收益相关 → 补偿已发生费用直接计入当期损益，补偿未来费用确认为递延收益

### CAS 30 财务报表列报

- 资产负债表
- 利润表
- 现金流量表（必须）
- 所有者权益变动表
- 附注（详细披露）

## 与小企业会计准则的核心差异对照

| 项目 | 小企业会计准则 | 企业会计准则 |
|------|---------------|-------------|
| 资产减值 | 不计提 | 必须计提 |
| 坏账准备 | 实际发生时核销 | 预期信用损失模型 |
| 递延所得税 | 不使用 | 必须确认 |
| 公允价值 | 不使用 | 部分资产适用 |
| 长期股权投资 | 统一成本法 | 区分成本法/权益法 |
| 政府补助 | 直接计入收入 | 区分资产/收益相关 |
| 收入确认 | 发出商品+收款权利 | 五步法模型 |
| 现金流量表 | 可选 | 必须 |
| 会计差错更正 | 未来适用法 | 追溯重述法 |

# Frontend Design Skill

> 来源: [anthropics/skills](https://github.com/anthropics/skills) (16.6k forks)
> 适配: 岛主 DaoZhu 平台（纯 HTML/CSS/JS，无 Node 依赖）

---

## 描述

创建独特的、生产级的前端界面，具有高设计质量。当用户要求构建 Web 组件、页面、
应用程序时使用此技能。生成有创意的、精致的代码和 UI 设计，避免千篇一律的 AI 美学。

---

## 适用场景

- 构建工作区前端页面
- 设计 landing page / dashboard
- 美化现有 UI
- 创建 HTML/CSS 组件

---

## 设计思维

编码前，理解上下文并确定一个**大胆的**美学方向：

- **目的**: 这个界面解决什么问题？谁在使用？
- **调性**: 选择一个极端方向：极简主义、复古未来、有机自然、奢华精致、
  趣味玩具、编辑杂志风、粗野主义、装饰艺术、柔和粉彩、工业实用等。
- **约束**: 技术要求（纯 HTML/CSS/JS，无框架，≤500 行/文件）
- **差异化**: 什么让这个设计令人难忘？用户会记住什么？

**关键**: 选择一个清晰的概念方向并精确执行。大胆的极繁主义和精致的极简主义
都可以——关键是意图性，而非强度。

---

## 前端美学指南

### 排版 Typography

- 选择美观、独特、有趣的字体
- 避免通用字体（Arial、Inter、Roboto、系统字体）
- 用一个独特的展示字体搭配精致的正文字体
- 中文推荐：思源黑体、霞鹜文楷、站酷系列

### 色彩与主题 Color & Theme

- 承诺一个连贯的美学
- 使用 CSS 变量保持一致性
- 主色 + 锐利强调色 > 胆怯的均匀分布色板
- 支持亮色/暗色主题切换

### 动效 Motion

- 用动画增强效果和微交互
- 优先使用纯 CSS 方案
- 聚焦高影响力时刻：一个精心编排的页面加载 + 交错显示
  比散落的微交互更令人愉悦
- 使用滚动触发和悬停状态制造惊喜

### 空间构图 Spatial Composition

- 意想不到的布局
- 不对称、重叠、对角线流动
- 打破网格的元素
- 慷慨的负空间或受控的密度

### 背景与视觉细节

- 创造氛围和深度，而非默认纯色
- 添加与整体美学匹配的上下文效果和纹理
- 渐变网格、噪点纹理、几何图案、层叠透明度、
  戏剧性阴影、装饰边框、自定义光标、颗粒叠加

---

## 禁止事项

**绝不**使用以下通用 AI 生成美学：

- ❌ 过度使用的字体（Inter, Roboto, Arial）
- ❌ 陈词滥调的配色（紫色渐变 + 白色背景）
- ❌ 可预测的布局和组件模式
- ❌ 缺乏上下文特色的千篇一律设计
- ❌ 跨生成收敛到相同选择

---

## 岛主平台适配规则

1. **纯 HTML/CSS/JS** — 不使用 React/Vue/Node
2. **文件 ≤ 500 行** — 超限按模块拆分
3. **CSS 变量** — 使用平台统一的 CSS 变量体系
4. **响应式** — 320px ~ 1920px 全覆盖
5. **可访问性** — 语义化 HTML，键盘可导航，ARIA 标签
6. **中文优先** — 字体、间距、行高适配中文排版
7. **主题兼容** — 必须支持亮色/暗色主题

## 输出格式

生成的前端代码应包含：

```
frontend/
├── index.html          # 页面结构（语义化）
├── css/
│   ├── base.css        # 变量 + 重置
│   └── components.css  # 组件样式
└── js/
    └── app.js          # 交互逻辑
```

每个文件开头注明用途，关键样式加注释说明设计意图。

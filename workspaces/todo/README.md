# 个人待办 📋

> 简洁高效的任务管理，让每一天都有条不紊。

---

## 介绍

个人待办是岛主平台的默认工作区之一，参考 [Super Productivity](https://github.com/johannesjo/super-productivity)（⭐ 18.8k）的核心任务管理逻辑，用 FastAPI + SQLite + 纯 HTML/JS 重写。

专注于个人日常任务管理，不做团队协作、不做时间追踪，只做好"待办"这一件事。

## 目标

- 目标 1：快速创建和管理任务（CRUD + 子任务）
- 目标 2：通过项目和标签灵活分类
- 目标 3：今日聚焦视图，帮助用户专注当天任务
- 目标 4：优先级标记，区分轻重缓急

## 功能列表

- [x] 任务增删改查
- [x] 任务状态流转（待办 → 进行中 → 已完成）
- [x] 四级优先级（紧急/高/中/低）
- [x] 项目分类（收件箱/工作/生活）
- [x] 今日聚焦视图
- [x] 截止日期 + 过期提醒
- [x] 子任务支持
- [x] 标签系统
- [ ] 拖拽排序（后续迭代）
- [ ] 番茄钟集成（可单独做工作区）

## 注意事项

- 数据存储在本地 `data.db`，请勿手动修改
- 端口 7801，如冲突请在 workspace.json 中修改
- 首次启动自动初始化数据库和默认项目
- 使用项目主环境 `.venv`，无需单独创建

# 财务助手 💰

> 公司财务信息管理：科目、凭证、账簿、报表

---

## 介绍

财务助手是岛主平台的默认工作区，提取自 AccoBot 项目的核心财务管理逻辑。
支持多公司账套管理、会计科目维护、记账凭证录入，由岛主 AI 管家辅助操作。

## 目标

- 目标 1：多公司账套管理
- 目标 2：会计科目表维护（支持树形结构）
- 目标 3：记账凭证录入（借贷平衡校验）
- 目标 4：科目余额查询

## 功能列表

- [x] 公司创建/管理
- [x] 会计科目 CRUD + 搜索
- [x] 凭证录入（含分录，借贷平衡校验）
- [x] 凭证状态流转（草稿→过账→作废）
- [x] 科目余额计算
- [ ] 财务报表生成（后续迭代）
- [ ] 期末结转（后续迭代）

## 注意事项

- 数据存储在本地 data.db
- 端口 7803
- 凭证必须借贷平衡才能保存
- 只有草稿状态的凭证可以删除

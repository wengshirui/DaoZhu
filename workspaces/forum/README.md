# 岛主论坛 🏝️

> 岛民交流，分享经验 · 对接 Gitee Issues

---

## 介绍

岛主论坛是平台的社区工作区，直接对接 Gitee 项目的 Issues 作为论坛服务。
用户可以在岛主内浏览、提问、回复，无需离开平台。

## 目标

- 目标 1：浏览 Gitee Issues（支持离线缓存）
- 目标 2：发表评论和创建新帖
- 目标 3：作为用户反馈渠道

## 功能列表

- [x] 浏览 Issues 列表（开放/已关闭）
- [x] 查看 Issue 详情和评论
- [x] 发表评论（需 Gitee Token）
- [x] 创建新 Issue（需 Gitee Token）
- [x] 本地缓存（离线可浏览历史）

## 注意事项

- 浏览不需要 Gitee 账号
- 发帖/回复需要在 .env 中配置 GITEE_TOKEN
- 获取 Token: Gitee → 设置 → 私人令牌 → 新建
- 端口 7802
- 数据缓存在本地 data.db

## 技术信息

| 项目 | 值 |
|------|-----|
| 端口 | 7802 |
| 数据库 | data.db (SQLite, 缓存) |
| 来源 | Gitee Issues API |
| 版本 | 1.0.0 |

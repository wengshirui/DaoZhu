# AccoBot 💰 智能财务助手

AccoBot 是一个 AI 驱动的财务助手，帮你轻松搞定记账、报税、对账、看报表等日常财务工作。

**不需要你懂会计** — AI 负责专业判断，你只需要用大白话告诉它你要做什么。

## 能做什么

- 📝 **记账** — 告诉 AI "今天付了3000元房租"，自动生成会计分录
- 🔍 **查账** — 问 AI "银行存款还有多少"、"这个月花了多少钱"
- 🏦 **对账** — 导入银行流水，AI 自动匹配核对
- 📊 **报表** — 一句话生成资产负债表、利润表
- 💡 **分析** — AI 主动告诉你钱花在哪了、哪里有异常
- ⚠️ **风控** — 自动检查每笔操作是否合规，提示风险
- 📚 **教学** — 遇到不懂的，AI 会解释并记录你的学习进度

## 快速开始

### 1. 安装

确保电脑上有 Python 3.11 或更高版本，然后：

```bash
# 下载项目
git clone https://github.com/your-repo/AccoBot.git
cd AccoBot

# 创建环境并安装
uv venv .venv --python 3.11
.venv\Scripts\activate
uv pip install -e ".[web]"
```

### 2. 启动

```bash
python -m accobot.web.server
```

浏览器会自动打开 `http://127.0.0.1:9120`。

### 3. 配置

首次打开会弹出设置页面，填入你的 AI 模型 API Key 即可开始使用：

- **推荐使用 DeepSeek**（便宜好用）：去 [DeepSeek 控制台](https://platform.deepseek.com/api_keys) 注册获取
- 也支持 OpenAI 或其他兼容接口

填好 Key，点保存，就可以开始对话了。**全程不需要编辑任何配置文件。**

## 使用示例

```
你：帮我记一笔，今天付了5000元房租给XX物业
AI：已创建凭证草稿：
    日期：2026-05-20
    摘要：付房租
    借：管理费用-租赁费  5,000.00
    贷：银行存款        5,000.00
    确认保存吗？

你：确认
AI：✅ 凭证已保存（记-2026-001）

你：这个月管理费用花了多少？
AI：本月管理费用合计 12,350.00 元，明细如下：
    - 租赁费：5,000.00
    - 办公费：3,200.00
    - 差旅费：4,150.00
```
